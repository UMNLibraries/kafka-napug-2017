Questions we will want to answer:

  • How has data changed over time?
  • What data did we put in Pure? When?
  • How do we reproduce errors?
  • Where did bad data come from? When?
  • How did we transform any given data?

Why Apache Kafka?

  • Permanent storage of streaming messages.
  • Start and stop reading messages at any offset, even in reverse.
  • Evolvable schemas for each message topic.
  • Connectors for common databases and other data tools.
